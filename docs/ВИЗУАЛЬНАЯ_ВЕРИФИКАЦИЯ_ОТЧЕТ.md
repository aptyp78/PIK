## 🌐 Веб-Интерфейс

### Доступ
- **URL:** http://localhost:8001
- **Запуск:** `python pik_visualization_server.py`
- **Автозапуск:** `./start_visualization.sh`

### Функции
1. **📊 Обзор** - Сводная статистика и графики
2. **🎯 Фреймворки** - Детальный анализ каждого фреймворка
3. **📈 Анализ** - Сетевая визуализация связей
4. **💡 Инсайты** - Ключевые находки и рекомендации

### API Эндпоинты
- `GET /api/summary` - Сводная информация
- `GET /api/frameworks` - Список фреймворков
- `GET /api/frameworks/{id}` - Детали фреймворка
- `GET /api/frameworks/{id}/network` - Данные для сетевой визуализации
- `GET /api/frameworks/{id}/download/json` - Скачать JSON
- `GET /api/frameworks/{id}/download/drawio` - Скачать Draw.io
